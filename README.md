# Hexlet Weather

Простое веб-приложение для получения текущего прогноза погоды по названию города (с поддержкой уточнения страны).

## Стек

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** отсутствует (клиент делает запросы напрямую к API OpenWeatherMap)
- **DB:** отсутствует

## Как запустить локально

1. Склонируйте или скачайте репозиторий.
2. Откройте `index.html` в браузере (двойной клик или через локальный сервер).

## Деплой

Проект можно увидеть на сайте https://hexlet-weather.vercel.app/
